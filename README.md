# ITINFOSTAR_Spam_Detector
Spam Message Detector (Java)
📌 Overview

This project is a Spam Message Detector built using core Java concepts. It analyzes a given message and determines whether it is spam based on predefined keywords and logic.

🚀 Features

Detects spam messages using keyword matching

Calculates spam score (%)

Displays word frequency analysis

Menu-driven program

Handles invalid/empty input using custom exception

🛠️ Technologies Used

Java (Core)

Concepts:

Strings

Arrays

Constructors

Exception Handling

⚙️ How It Works

User enters a message

Program splits the message into words

Compares words with predefined spam keywords

Calculates spam count and percentage

Displays result as:

⚠️ Spam Detected

✅ Safe Message

▶️ Sample Output
Enter your message: You won a free prize now

--- Word Analysis ---
you -> 1
won -> 1
a -> 1
free -> 1
prize -> 1
now -> 1

Spam Score: 33.33%
⚠️ SPAM DETECTED!
